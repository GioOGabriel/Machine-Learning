# Projetos de Machine Learning 🤖

Bem-vindo ao meu repositório de Machine Learning! Este espaço é dedicado aos meus projetos acadêmicos e pessoais com foco em Ciência de Dados, Inteligência Artificial e Redes Neurais.

## 📌 Projeto em Destaque
* **Predição de Alzheimer:** Um modelo de rede neural desenvolvido em Python para prever a probabilidade de diagnóstico de Alzheimer com base em dados clínicos do Kaggle.

## 🛠️ Tecnologias Utilizadas
* Python
* Pandas & NumPy
* Scikit-Learn
* Deep Learning / Redes Neurais
